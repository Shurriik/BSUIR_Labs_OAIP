BJCH ПРОГРАММА - ИНСТРУКЦИЯ ПО УСТАНОВКЕ

1. УСТАНОВКА:
   - Скачайте папку с программой
   - Запустите файл "install.bat"
   - Следуйте инструкциям на экране

2. ЗАПУСК:
   После установки программа запустится автоматически.
   Дальше можно запускать через:
   - Ярлык на рабочем столе "BJCH Программа"
   - Меню Пуск -> Программы -> BJCH

3. СТРУКТУРА ПРОГРАММЫ:
   - main.exe / main.py - главная программа
   - assets/ - папка с изображениями
   - data/ - папка с данными
   - styles.qss - файл стилей

4. УДАЛЕНИЕ:
   Для удаления запустите файл "uninstall.bat"

ТРЕБОВАНИЯ:
- Windows 7/8/10/11
- Для .py версии: Python 3.6+
